SanAlejo App

Tecnología
- React Native (Expo)
- SQLite (expo-sqlite)

Instalación
1. Clonar repositorio:
git clone https://github.com/JuanDNeira/SanAlejo_JuanNeira.git

2. Instalar dependencias:
npm install

3. Ejecutar:
npx expo start

Funcionalidades
- Crear contenedores
- Agregar objetos a contenedores
- Eliminar contenedores y objetos
- Persistencia local con SQLite
